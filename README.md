# meteor-study
this is a repo to recarding meteor

## 这是一个利用meteor框架搭建类似hackernews的实时性的应用
## 怎么运行:
* 官网下载安装 meteor (www.meteor.com)
* 然后git clone https://github.com/cllgeek/meteor-study.git
* cd hackernews
* 在命令行 运行 meteor 就可以了,浏览器访问http://localhost:3000/
